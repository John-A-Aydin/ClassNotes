#### Ordered Pair
$\large (u,v) = (a,b)\text{ if and only if } u=a\text{ and }b=b$


$\Large A\times B = \{(u,v)|u\in A, v\in B\}$

$\large \{1,2,3\}\times\{3,4\} = \{(1,3), (1,4), (2,3),(2,4),(3,3),(3,4)\}$ 

### Ordered k-tuple
$\Large (a_1,a_2,a_3\text{, ... }a_k)$
$|A_1, A_2, ... A_k| = |A_1|*|A_2|*...*|A_k|$ 

### Cardinality: 

If
$A\times B = C$
then
$|C| = |A||B|$
