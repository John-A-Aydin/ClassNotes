A wedge's volume is defined as
$$\large \int_{0}^{3}\int_{0}^{10-x^{2}}\int_{0}^{3-x}  \; d z \; d y \: d x$$
$\large 0 \le z \le 3-x$
$\large 0 \le y \le 10-x^{2}$
$\large 0 \le x \le 3$

Switch the order to dy dx dz

$\large 0 \le z \le 3$
$\large 0 \le y \le 10 - x^{2}$
$\large 0 \le x \le z-3$

$$\large \int_{0}^{3}\int_{0}^{z-3}\int_{0}^{10-x^{2}}  \; d y \; d x \: d z$$
